# NX 2306 - Get Material from Section View
## VB Code Implementation Guide

---

## Overview
This code retrieves material information for all components visible in section views of NX drawings, optimized for NX 2306.

---

## NX 2306 Specific Features

### 1. **Updated Namespaces**
- `Imports NXOpen.Drawings` - New in recent NX versions for better drawing support
- `Imports NXOpen.Assemblies` - For component assembly access
- Fully compatible with NX 2306 API

### 2. **DrawingSheet and DrawingSheetView**
In NX 2306:
```vb
Dim drawingPart As DrawingSheet = TryCast(workPart, DrawingSheet)
Dim sectionViews As DrawingSheetView = drawingPart.DrawingViews
```

### 3. **Component Visibility Check**
```vb
If comp.Visible Then
    visibleComponents.Add(comp)
End If
```

### 4. **Material Attribute Handling**
NX 2306 supports multiple attribute types:
- **StringAttribute** - For text material names
- **IntegerAttribute** - For material codes/IDs
- **RealAttribute** - For material properties (density, etc.)

## How Material is Typically Stored in NX

### Option A: Part Attributes (Most Common)
Material is stored as a custom attribute on the part:
```
Part > Properties > Attributes > Material = "Steel"
```

### Option B: User-Defined Attributes
```
Right-click Part > Properties > Custom User Attributes > Material = "Aluminum"
```

### Option C: Part Description
```
Part Name: "Bracket_Steel_v01"
Description: "Bracket component made from Steel"
```

### Option D: Mass Properties Database
```
Density = 7850 kg/m³ (Steel)
Density = 2700 kg/m³ (Aluminum)
```

## Key Code Functions

### 1. **ProcessSectionView()**
- Identifies section views in the drawing
- Gets the source view (main view being sectioned)
- Retrieves the origin model (assembly)
- Collects all visible components

### 2. **GetVisibleComponentsInSection()**
- Recursively traverses component assembly tree
- Filters for visible components
- Returns list of Component objects

### 3. **ExtractComponentMaterial()**
- Calls 4 different methods to find material
- Tries attributes, user attributes, description, and mass properties
- Handles errors gracefully

### 4. **Material Extraction Methods**
1. `GetMaterialFromAttributes()` - Standard NX attributes
2. `GetMaterialFromUserAttributes()` - Custom user attributes
3. `GetMaterialFromDescription()` - Part name/description analysis
4. `GetMaterialFromMassProperties()` - Density-based lookup

## Troubleshooting

### Issue: "No components visible in this section"
- **Cause**: Components might not have visibility flag set
- **Fix**: Uncomment/modify the visibility filter in `GetVisibleComponentsInSection()`

### Issue: "Material: Not found"
- **Cause**: Material not stored as expected
- **Fix**: Check where your system stores material data
- **Solution**: Add custom attribute search in materialAttrs array

### Issue: "No origin model found"
- **Cause**: Section view not properly linked to source view
- **Fix**: Verify section view is created from main view correctly

### Issue: Drawing is not recognized
- **Cause**: Active part is not a drawing
- **Fix**: Make sure you have a drawing open, not just a part file
